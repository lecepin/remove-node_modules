# Remove node_modules
一键查找并删除 `node_modules` 文件夹。

One-click find and remove node_modules folder.

## 使用

下载 `rnm.php` 文件，执行：

```bash
$ php rnm.php someDir

# OR
$ chmod +x rnm.php
$ ./rnm.php someDir
```
